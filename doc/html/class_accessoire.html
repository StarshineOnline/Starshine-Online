Constructeur permettant la création d'un accessoire.

Les valeurs par défaut sont celles de la base de donnée. Le constructeur accepte plusieurs types d'appels: -Accessoire() qui construit un objet "vide". -Accessoire($id) qui va chercher l'objet dont l'id est $id dans la base. -Accessoire($nom,...,$forceReq) qui construit un nouvel objet à partir des valeurs.
