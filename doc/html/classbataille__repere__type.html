Constructeur permettant la création d'un repère de bataille.

Les valeurs par défaut sont celles de la base de donnée. Le constructeur accepte plusieurs types d'appels: -bataille_repere_type() qui construit un etat "vide". -bataille_repere_type($id) qui va chercher l'etat dont l'id est $id -bataille_repere_type($array) qui associe les champs de $array à l'objet.
