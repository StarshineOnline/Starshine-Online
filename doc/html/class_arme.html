Constructeur permettant la création d'une armure Les valeurs par défaut sont celles de la base de donnée.

Le constructeur accepte plusieurs types d'appels: -Arme() qui construit un objet "vide". -Arme($id) qui va chercher l'objet dont l'id est $id dans la base. -Arme($nom,...,$image) qui construit un nouvel objet à partir des valeurs.
