Constructeur permettant la création d'un etat de message.

Les valeurs par défaut sont celles de la base de donnée. Le constructeur accepte plusieurs types d'appels: -Objets() qui construit un etat "vide". -Objets($id) qui va chercher l'etat dont l'id est $id_bourse_royaume dans la base.
