Constructeur permettant la création d'un thread de messagerie.

Les valeurs par défaut sont celles de la base de donnée. Le constructeur accepte plusieurs types d'appels: -messagerie_thread() qui construit un thread "vide". -messagerie_thread($id) qui va chercher le thread dont l'id est $id dans la base.
