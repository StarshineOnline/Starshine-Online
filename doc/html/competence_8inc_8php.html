Cette fonction permet d'apprendre une compétence, gérant les prérequis, les grimoires, les taxes.

Paramètres:
ecole l'école/type de compétence à apprendre
id_competence l'id de la compétence à apprendre
joueur le joueur qui apprend
R le royaume à qui on paye la taxe
grimoire true si on apprend depuis un grimoire(gratis), false sinon
Renvoie:
true si on a pu apprendre la compétence, false sinon
