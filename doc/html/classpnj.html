Class PNJ.

Classe PNJ Classe représentant un PNJ et permettant de l'ajouter, le modifier ou le supprimer de la base.

Constructeur.

Constructeur.

Le constructeur peut être utilisé de plusieurs façons: -PNJ() qui crée un pnj "vide". -PNJ($id) qui récupère les informations du pnj dans la base. -PNJ($nom, $coord_x, $coord_y, $image, $texte) qui crée un nouveau pnj.

Paramètres:
$id int Id du pnj dans la base.
$nom String Nom du pnj.
$coord_x int Coordonnée en X du pnj.
$coord_y int Coordonnée en Y du pnj.
$image String L'adresse? de l'image du pnj.
$texte String
