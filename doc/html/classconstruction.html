Constructeur permettant la création d'une construction.

Les valeurs par défaut sont celles de la base de donnée. Le constructeur accepte plusieurs types d'appels: -construction() qui construit un etat "vide". -construction($id) qui va chercher l'etat dont l'id est $id -construction($array) qui associe les champs de $array à l'objet.

Réimplémentée dans bourg.
